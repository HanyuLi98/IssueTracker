# Changelog

## 2026-05-09（第二批）

### 新功能

#### 1. 行详情抽屉
- 点击任意表格行（避开按钮/输入框）从右侧滑出详情面板
- 面板显示所有字段，备注以完整时间线格式展示，每条以蓝色日期分隔
- 底部「复制记录文本」一键将记录内容复制到剪贴板
- 点击蒙层或右上角 ✕ 关闭

#### 2. 移动端适配
- 屏幕宽度 < 720px 时，表格自动切换为卡片列表，点击卡片同样打开详情抽屉
- 侧边栏在移动端隐藏，主区域左上角出现「☰ 菜单」入口，点击弹出覆盖层侧边栏
- 点击任意导航项或背景蒙层自动收起侧边栏

#### 3. 数据看板升级
- 月度趋势由柱状图改为 **SVG 折线图**，折点标注具体数值
- 新增 **甜甜圈饼图**，展示各类别记录数量及百分比
- 工程师工作量调整为全宽独立卡片，视觉更清晰
- 折线图与饼图在移动端自动切换为单列布局

#### 4. 货币显示修正
- 报价金额统一改为 `4,641 元` 格式，消除 ¥ 符号在不同系统/字体下的渲染歧义

#### 5. 更新失败问题修复
- 修复售前咨询、Ticket 双击编辑后保存失败的问题
- 根因：虚拟字段 `_hasKeyProject` 被错误传入 Supabase，导致 400 错误
- 通过 `TASK_DB_COLS` 白名单过滤，只发送数据库实际存在的字段

## 2026-05-09（第一批）

### 新功能

#### 1. 全局隐藏 / 显示已解决
- 将「隐藏已解决」按钮从每个独立表格移至顶部 Tab 栏
- 一次点击同时对售前咨询、售中、售后、Key Project 等所有分类生效
- 按钮状态全局同步，眼睛图标直观切换

#### 2. 新增记录支持连续天数
- AddModal 日期字段新增 **单天 / 连续天数** 切换
- 选择「连续天数」后显示起始日期和结束日期两个选择框
- 实时提示「共 N 天，每天创建一条记录」
- 保存时批量写入 Supabase，每天生成一条独立记录

#### 3. 备注自动时间戳
- **新增记录**：备注非空时自动在内容前加当天日期前缀（如 `5.9 安装完成`）
- **行内编辑备注**：双击备注单元格，弹出空输入框（placeholder: 添加备注更新...），保存后新内容以 `M.DD 内容` 格式追加到最前，原有内容保留在下方
- **显示效果**：备注列解析每行，日期前缀（如 `5.9`、`5.20`）高亮显示为蓝色，形成可读的更新日志时间线
