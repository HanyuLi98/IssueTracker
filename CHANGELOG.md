# Changelog

## 2026-05-09

### 新功能

#### 1. 全局隐藏 / 显示已解决
- 将「隐藏已解决」按钮从每个独立表格移至顶部 Tab 栏
- 一次点击同时对售前咨询、售中、售后、Key Project 等所有分类生效
- 按钮状态全局同步，眼睛图标直观切换

#### 2. 新增记录支持连续天数
- AddModal 日期字段新增 **单天 / 连续天数** 切换
- 选择「连续天数」后显示起始日期和结束日期两个选择框
- 实时提示「共 N 天，每天创建一条记录」
- 保存时批量写入 Supabase，每天生成一条独立记录

#### 3. 备注自动时间戳
- **新增记录**：备注非空时自动在内容前加当天日期前缀（如 `5.9 安装完成`）
- **行内编辑备注**：双击备注单元格，弹出空输入框（placeholder: 添加备注更新...），保存后新内容以 `M.DD 内容` 格式追加到最前，原有内容保留在下方
- **显示效果**：备注列解析每行，日期前缀（如 `5.9`、`5.20`）高亮显示为蓝色，形成可读的更新日志时间线
